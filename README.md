# 📊 Nepal Literacy Analysis (Census 2021)

This project analyzes literacy patterns in Nepal using census data (population aged 5+).  
It explores literacy distribution across regions, provinces, and gender.

---

## 🔍 Project Overview

The goal of this project is to understand:

- Literacy vs Illiteracy in Nepal  
- Regional differences (Mountain, Hill, Tarai)  
- Province-level literacy patterns  
- Gender-based literacy comparison  

---

## 📊 Key Results

- 📈 Total Literate Population: **26,725,295**  
- 📉 Total Illiterate Population: **20,377,980**  

- 🏆 Highest Literacy Province: **Bagmati**  
- ⚠️ Lower Literacy observed in **Mountain region**

---

## 📌 Analysis Performed

### 1. Data Cleaning
- Removed hierarchical duplicates (Nepal, regions, provinces)
- Converted text data into numeric format
- Handled missing values

### 2. Regional Analysis
- Compared literacy across Mountain, Hill, and Tarai
- Identified geographical disparities

### 3. Province Analysis
- Compared literacy vs illiteracy across 7 provinces
- Identified top-performing regions

### 4. Gender Analysis
- Compared literacy levels between male and female populations

---

## 🧠 Key Insights

- Literacy is higher than illiteracy overall in Nepal  
- Bagmati Province shows the highest literacy level  
- Mountain region has comparatively lower literacy rates  
- Significant disparities exist across provinces and regions  
- Gender gap in literacy still exists but is not extreme
